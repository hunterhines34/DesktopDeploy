Desktop Deploy

A simple tool designed to download software and configure new PCs from a single controlled repository.

Desktop Deploy (DD) is an all-in-one tool designed for desktop technicians to download software, obtain computer diagnostic data and also run common scripts from one single location.

Common DD use cases

  • Download bundles of software from a single repository
  • Download bundles of scripts from a single location
  • Access commonly used PC diagnostics
